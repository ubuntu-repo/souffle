Detailed Description

template<>
class souffle::Trie< 1u >

A template specialization for tries containing tuples exhibiting a single element.

For improved memory efficiency, this level is the leaf-node level of all tires exhibiting an arity >= 1. Internally, values are stored utilizing sparse bit maps.
