Detailed Description

template<typename Iter, typename Index>
class souffle::ram::iterator_utils::filter_iterator< Iter, Index >

A generic iterator realizing a range query utilizing a scan, by filtering out invalid elements.

This iterator is utilized in cases where no index supporting a query is available.
