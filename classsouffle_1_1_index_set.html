Detailed Description

Computes the index set for a relation.

If the indexes of a relation can cover several searches, the minimal set of indexes is computed by Dilworth's problem. See

"Optimal On The Fly Index Selection in Polynomial Time" https://arxiv.org/abs/1709.03685
